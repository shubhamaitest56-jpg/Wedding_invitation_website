# Tanmay Weds Tanya - Wedding Invitation

Welcome to the digital wedding invitation for Tanmay and Tanya! This premium, interactive wedding website features:

- A luxurious "Divine Reveal" opening sequence
- Scratch-to-reveal Save the Date cards
- Live countdown timer
- Interactive event schedule
- Our Story timeline
- RSVP integration
- Ambient music player
- High-end cinematic animations powered by Framer Motion

## Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [React](https://react.dev)
- [Framer Motion](https://www.framer.com/motion/)
- [Tailwind CSS](https://tailwindcss.com) (via standard CSS modules & utility classes)
